GalGame. Redifined with GenAI. </>

# Usage

1. Clone the respositry.
2. Config `OPENAI_API_KEY` , `OPENAI_BASE_URL` and `LLM_MODEL` in `.env`.
3. Install the requirements.
```bash
pip install -r ./requirements.txt
```
4. Run Gradio Server.
```bash
python gradio_server.py
```
